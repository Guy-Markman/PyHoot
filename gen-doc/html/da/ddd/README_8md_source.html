1 **PyHoot**
2 
3 Kahoot clone based on Python, final project for Gvahim.
4 
5 Warning:
6 Python 2.7 must be installed on the system.
7 if you don't have it have it, you can download it from here. [Link](https://www.python.org/ftp/python/2.7.13/python-2.7.13.msi)
8 
9 **When you install in `Customize Python 2.7.x`, make sure you click on `Add python.exe to Path` and then on `Will be installed on local hard drive`.**
10 
11 How to start the server:
12 
13 If you are using windows (recommended for Windows users):
14 
15 Go to the PyHoot directory and click on `Start Server (Windows).bat` from the PyHoot Directory.
16 It will start the server on your IP address with port 80, meaning that you can join the game from any device in the same network as your server by writing the address in the address line.
17 If you don't know your own IP address you can see it on the third line on the window that will be opened.
18 
19 If you are using command line \ shell (recommended for experts and Linux users):
20 
21 Start the command line or shell in your system and change the directory where you put the directory of files (named PyHoot).
22 Write the next command in the command line \ shell:
23 
24 ```python -m PyHoot```
25 
26 This command will enable you to use different arguments than the default ones, you can do it using the next script.
27 
28 ```python -m PyHoot --[name of argument]=[the value of the arguement]```
29 In this way, you'll be able to add as many arguments as you want.
30 
31 Arguments list:
32 
33  -h, --help show this help message and exit
34  --address ADDRESS [ADDRESS ...]
35  The address(es) we will connect to, default
36  ['localhost:8080']
37  --buff-size BUFF_SIZE
38  Buff size for each time, default 1024
39  --base BASE Base directory
40  --io-mode {select, poll}
41  IO that will be used, default select.
42  In windows only select available.
43  --log-level {DEBUG,INFO,WARNING,CRITICAL,ERROR}
44  Log level
45  --log-file FILE Logfile to write to, otherwise will log to console.
